# 贡献指南

感谢您对本项目的关注！本仓库是 [vorniches/tapnote](https://github.com/vorniches/tapnote) 的一个增强版本。

## 🤔 我应该向哪个仓库贡献？

### 向原项目贡献（vorniches/tapnote）

如果您的改进是**通用的、核心功能的改进**，建议向原项目提交 PR：
- Bug 修复
- 核心功能改进
- 文档改进
- 测试改进
- UI/UX 优化

### 向本 Fork 贡献（zoidberg-xgd/tapnote）

如果您的改进是**特定功能的增强**，欢迎向本仓库提交 PR：
- 评论系统相关功能
- 数据迁移功能
- PythonAnywhere 部署相关
- 特定工具脚本
- 中文文档

## 📝 提交 Pull Request

1. **Fork 本仓库**
2. **创建功能分支**：`git checkout -b feature/your-feature-name`
3. **提交更改**：`git commit -m "Add: your feature description"`
4. **推送到您的 Fork**：`git push origin feature/your-feature-name`
5. **创建 Pull Request**

## 🧪 测试

在提交 PR 前，请确保：
- 运行测试：`./run_tests.sh`
- 检查代码风格
- 更新相关文档

## 📋 提交信息规范

- `Add:` 新功能
- `Fix:` Bug 修复
- `Update:` 功能更新
- `Docs:` 文档更新
- `Refactor:` 代码重构
- `Test:` 测试相关

## ❓ 问题反馈

如果遇到问题或有建议，请：
1. 先搜索已有的 Issues
2. 如果没有，创建新的 Issue
3. 提供详细的问题描述和复现步骤

## 📄 许可证

贡献的代码将使用与原项目相同的 MIT 许可证。
